# Personal Portfolio Website

This is a personal portfolio website built with Next.js and Tailwind CSS, designed to showcase my skills, projects, and professional profile. The site is also integrated with Sanity.io for content management, Framer Motion and Animate.css for smooth animations, NextUI for UI components, and Resend for handling email functionality.

## 🚀 Tech Stack

- **Next.js** - React framework for fast, server-rendered applications.
- **Tailwind CSS** - Utility-first CSS framework for styling.
- **Sanity.io** - Headless CMS for managing content.
- **Framer Motion** - Library for animations and interactions.
- **Animate.css** - Library for CSS animations.
- **NextUI** - UI components for a seamless interface.
- **Resend** - API for handling email.

## 📸 Features

- **Responsive Design**: Optimized for desktop and mobile devices.
- **Dynamic Content**: Easily update content through Sanity.io CMS.
- **Smooth Animations**: Engage users with Framer Motion and Animate.css.
- **Email Contact Form**: Integrated with Resend for email delivery.
- **Modern UI**: Clean, user-friendly interface with NextUI.

## 🛠️ Installation and Setup

To run this project locally, follow these steps:
1) git clone "name of the repo"
2) yarn install
3) yarn run dev

### Prerequisites

- Node.js and npm installed
- Sanity.io project set up (for managing portfolio content)
- Resend account for email integration
